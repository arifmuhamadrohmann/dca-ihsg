Komponen breakdown

  • Header Header.tsx
    Title + ticker subtitle + ikon Info & GitHub (link ke repo).
  • Input panel InputPanel.tsx
    Nominal (currency input, format Rp ribuan), Start date (MonthYearPicker), End date (default "Hari ini" toggle).
  • Summary card SummaryCard.tsx
    Hero metric (Nilai portofolio) + delta absolut & % + grid 2x2 stats (Total setoran, XIRR, Bulan investasi, Periode). Warna hijau saat positif, merah saat negatif. Card ini target screenshot.
  • Chart PortfolioChart.tsx
    Line chart 2 series (Recharts). Series solid hijau = nilai portofolio, dashed gray = total setoran kumulatif. Crisis markers sebagai vertical reference lines (1998, 2008, 2013, 2015, 2020).
  • Actions ActionsBar.tsx
    Simpan PNG (html-to-image, capture summary+chart) + Bagikan (Web Share API, fallback copy link).
  • Methodology accordion MethodologyAccordion.tsx
    Collapsed default. Saat expand: data source, asumsi, formula, batasan.
  • Disclaimer Disclaimer.tsx
    Always visible. "Bukan rekomendasi investasi".
  • Footer
    Data freshness label + tanggal update terakhir.

Behavior notes

  • Saat user adjust input apapun, summary + chart update real-time (debounced 200ms).
  • Loading state: skeleton di summary card & chart saat hitung pertama atau saat data fetch.
  • Tap "Hari ini" toggle di End date → kembali ke tanggal terakhir tersedia, abu-abu kalau aktif.
  • Chart responsive — di desktop bisa lebih lebar tapi tinggi tetap proporsional.
  • Tap PNG → generate image 1080x1080 berisi summary card + chart + branding kecil.
  • Share button → kalau Web Share API tidak tersedia, fallback ke copy URL ke clipboard + toast.

Desktop adaptation

  • Container max-width 720px, center aligned. Tidak butuh layout dua kolom — single column tetap optimal untuk konteks "telling a story".
  • Chart height bertambah jadi ~280px untuk readability.
  • Input panel bisa horizontal (3 fields satu baris) di breakpoint >640px.

Empty / error states

  • Start date di masa depan → red helper text, summary blank dengan placeholder dash.
  • Data unavailable → fallback ke last commit JSON + banner "Data terakhir di-update [tanggal]".
  • Periode < 1 bulan → warning "Periode terlalu pendek untuk simulasi DCA".
